# Audio Credits

UI sound effects are selected from Kenney Interface Sounds.

- Source: https://kenney.nl/assets/interface-sounds
- License: Creative Commons CC0
- Files used:
  - `ui-click.ogg`
  - `ui-select.ogg`
  - `ui-open.ogg`
  - `ui-open-soft.wav` (local softer replacement for the primary open action)
  - `ui-reveal-soft.wav` (local softer cue for generated results)
  - `ui-success.ogg`
  - `ui-error.ogg`

The app falls back to short Web Audio tones when a browser cannot play the audio files.
